[[inter_freeze]]
= freeze

tools to generate quantities from solvent after equilibration.
also used to remove objects colliding with the newly created wall

[source, cpp]
----
include::{src}/inter/freeze/imp.h[tag=int]
----
<1> create rigid bodies, remove rigid bodies and rbcs colliding with
    the walls and remove solvent at rigid bodies locations

To avoid long argument list, the data is packed in helper structures:

[source, cpp]
----
include::{src}/inter/freeze/imp.h[tag=helpers]
----
