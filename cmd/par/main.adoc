= u.par(1)

== NAME
u.par - create key=value configuration file


== SYNOPSIS
*u.par* ['OPTIONS'] ['FILE']


== DESCRIPTION
The u.par(1) write udx parameter file to stdout

== OPTIONS
*-w::
  Overwrite parameters set in the 'FILE'

*key=val*::
  Append key=val at the end of the output

== EXAMPLES
`u.par a=1`

`u.par a=1 b=2 c=\"cell.off\" input.file`
